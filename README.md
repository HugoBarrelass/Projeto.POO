## ▶️ Como Executar

### Pré-requisitos

- Java 17
- Maven

### Executar o jogo

Abrir a pasta `ProjetoPOO` no IntelliJ IDEA e executar o ficheiro:

`src/main/java/pt/poo/towerdefense/App.java`

### Executar os testes unitários

```bash
cd ProjetoPOO
mvn test
```
